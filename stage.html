Stages

Stage 1 – Développement d’une application web interne (projet encadré)

Projet : Dans le cadre de mon stage, il m’a été confié un projet pratique visant à digitaliser une activité commerciale à l’international. J’ai développé, à titre d’entraînement encadré, une application web interne pour simplifier la gestion des commandes à l’export. L’application permet de sélectionner un produit, une destination et d’obtenir automatiquement une estimation des coûts ainsi qu’un devis en PDF. Ce projet m’a permis de mettre en pratique mes compétences en développement web dans un contexte proche du réel.


Compétences acquises :

  • Développement full-stack d’une application web légère
  • Conception d’une interface simple et ergonomique (formulaire de sélection)
  • Implémentation de la logique de calcul des coûts (volume, transport, emballage)
  • Génération dynamique de documents PDF (devis, bons de commande)
  • Structuration du code et gestion de versions avec Git
  • (Optionnel) Mise en place d’une base de données simple pour gérer les produits et tarifs

Technologies utilisées :

  • Python
  • Flask
  • HTML / CSS (Bootstrap)
  • WeasyPrint ou ReportLab
  • SQLite ou structure JSON
  • Visual Studio Code, Git/GitHub

Stage 2 – Développement d’une application de gestion d’appels d’offres (projet encadré)

Projet : Dans le cadre de mon stage, il m’a été confié un projet pratique à titre d’entraînement pour développer une application web interne de gestion des appels d’offres. Cette application permet de centraliser les demandes, suivre les offres et générer des documents liés aux appels d’offres, afin de mettre en pratique mes compétences en développement web dans un contexte proche du réel.


Compétences acquises :

  • Développement web full-stack avec Java et Spring Boot
  • Conception d’une interface utilisateur ergonomique avec Bootstrap
  • Gestion des interactions avec la base de données et intégration back-end
  • Structuration et organisation du code dans un projet encadré
  • Pratique du suivi de projet en environnement simulé

Technologies utilisées :

  • Java, Spring Boot
  • HTML / CSS / Bootstrap
  • Base de données (H2 ou MySQL selon le projet)
  • Visual Studio Code ou IntelliJ IDEA
  • Git / GitHub pour le contrôle de version
