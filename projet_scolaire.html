Projets scolaires

Application de gestion des appels d’offres

Dans le cadre de ma formation, j’ai développé une application en Java permettant de gérer des appels d’offres. Les utilisateurs peuvent s’inscrire, se connecter, consulter des appels d’offres et proposer des offres. Les entreprises peuvent publier des appels d’offres et consulter les propositions reçues. L’interface a été réalisée avec JavaFX et FXML, offrant une utilisation simple et intuitive.

Technologies utilisées :

  • Java
  • JavaFX / FXML
  • MySQL
  • IntelliJ IDEA

Application web de réservation (RESA YVA)

J’ai développé une application web de réservation d’hébergements. Les utilisateurs peuvent se connecter, consulter les hébergements, visualiser les disponibilités par semaine et effectuer des réservations. Le gestionnaire dispose d’un espace administrateur pour gérer les réservations et administrer les hébergements.

Technologies utilisées :

  • PHP
  • HTML / CSS
  • JavaScript
  • SQL / MySQL
  • EasyPHP
  • Visual Studio Code

Application web de gestion d’un zoo

J’ai développé une application web en PHP permettant de gérer un zoo connecté à une base de données. L’application permet d’ajouter, modifier et supprimer des animaux. Deux types de comptes sont disponibles : employé (gestion des animaux) et administrateur (gestion des animaux et des employés).

Technologies utilisées :

  • PHP
  • HTML / CSS
  • SQL / MySQL
  • EasyPHP
  • Visual Studio Code
